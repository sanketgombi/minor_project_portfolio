Problem Statement

Identifying the core challenge and project goals

Introduction

This project focuses on designing and developing a custom end-effector for the TM5-700 cobot to automate pallet emptying and redocking into the raw material ASRS.

The system will integrate with the LD-250 AMR and trigger workflow transitions via OPC protocol after pallet placement in the finished goods ASRS.

Project Goals
  • 1 Pick pallets from ASRS using MoMa (Mobile Manipulator) (LD-250 AMR + TM5-700)
  • 2 Empty / unload materials from pallet
  • 3 Place empty pallets into finished goods ASRS
  • 4 Trigger next AMR–Cobot cycle automatically via OPC

System Architecture

Manipulator OMRON TM5-700 Cobot
Mobile Base OMRON LD-250 AMR
Communication OPC Protocol
Target System Raw Material ASRS

Key Performance Criteria

Our design solution is governed by a strict set of quantitative objectives to ensure industrial viability. The end-effector must operate seamlessly within the collaborative safety thresholds specified by ISO 15066.

  • Weight Minimization: End-effector total weight must remain under 1.2 kg to preserve at least 80% of the TM5-700's total 5.0 kg payload capacity.
  • Safe Force Limits: Grasp force feedback must limit gripping pressure under 15N to avoid deformation of thin-walled cardboard bins.
  • Precision Alignment: Integrated structural guides must guide the slot redocking trials to within ±2.0 mm tolerance range.
  • Direct Integration: Control logic must execute natively via Modbus TCP command sets through the OMRON TM Flow camera-interface scripts.
💡

The Solution Concept

A custom 3D-printed hybrid compliance gripper that combines rigid structural links with soft TPU finger tips and pressure sensors to address all of these performance criteria within a cost-effective, local manufacturing bill of materials.
